## Conventions

- Special API routers (`dashboard`, `accounting`, `reports`, `warehouse`,
  `fleet`, `vehicleAccounting`, `vehicleDocs`) mount **before** the generic
  `/api/:resource` CRUD router in `server/index.ts` вЂ” order matters.
- All UI text is bilingual via `src/i18n/strings.ts` (+ per-module
  `strings.ts`) вЂ” add both AR and EN keys for any new copy.
- Each module lives under `src/modules/<name>/` and exports `routes.tsx`.
- Locked modules keep their URL paths but render `LockedPage`
  (`gate()` in `src/App.tsx`, `isModuleLocked()` in `src/config/nav.ts`).

## Invariants

- Journal entries must balance (ОЈdebit = ОЈcredit) вЂ” server rejects unbalanced
  entries.
- Warehouse transactions must actually move stock and be audit-logged
  (`server/lib/eventLog.ts`).
- Companies / projects / chart-of-accounts are real data вЂ” never delete,
  overwrite, or reseed `server/data` in a way that loses them.
- Locked modules (`UNLOCKED_MODULES` in `src/config/nav.ts`) must never
  crash вЂ” they always render `LockedPage`.
