# 💰 Module 05 — Accounting (المحاسبة)

---

## Overview
Full double-entry accounting system based on the company's official Chart of Accounts (الدليل المحاسبي). Every transaction page includes the required fields.

---

## 📄 Transaction Page Fields (Required on EVERY entry)

| Field | Arabic | Type |
|-------|--------|------|
| Serial Number | رقم التسلسلي | Auto-generated |
| Document Number | رقم الوثيقة | String |
| Company | الشركة | Select (11 companies) |
| Project | المشروع | Select (linked project) |
| Account | الحساب | Select (from chart) |
| Description | البيان | Text |
| Currency | العملة | Select (IQD/USD/EUR...) |
| Price/Rate | السعر | Number |
| Value | القيمة | Number (auto = qty × price) |
| Debit | مدين | Number |
| Credit | دائن | Number |
| Date | التاريخ | Date |
| Notes | ملاحظة (private) | Text |

## 📑 Accounting Reports

| Report | Arabic | Description |
|--------|--------|-------------|
| Trial Balance | ميزان المراجعة | All accounts with debit/credit totals |
| Income Statement | قائمة الدخل | Revenue - Expenses = Net Profit |
| Balance Sheet | الميزانية العمومية | Assets = Liabilities + Equity |
| Cash Flow | التدفقات النقدية | Operating/Investing/Financing |
| General Ledger | دفتر الأستاذ العام | All transactions per account |
| Journal | دفتر اليومية | Chronological all entries |
| Project P&L | ربح وخسارة المشروع | Per-project profitability |
| Company P&L | ربح وخسارة الشركة | Per-company profitability |
